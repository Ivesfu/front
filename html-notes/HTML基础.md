+ 标签的属性

  在标签（**开始标签或者自结束标签**）中可以设置属性，**属性是一个名值对**，用于设置标签中内容如何显示。属性和标签名或其他属性要用空格隔开，有些属性有属性值，有些属性则无属性值，属性值应该用引号引起来。

  ~~~html
  <!DOCTYPE html>
  <html>
      <head>
          <title>标签的属性</title>
      </head>
      <body>
          <h1><font color = "red">这是一个大标题</font></h1>
      </body>
  </html>
  ~~~

+ 文档申明（doctype）

  文档声明用来**告诉浏览器当前网页的版本**。`<!DOCTYPE html>`

  浏览器解析网页是自上向下逐行解析的。
