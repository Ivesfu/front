+ **HTML**----超文本标记语言

  word文档不是文本，只有在纯文本编辑器中写的内容才是文本。

  主要负责网页三元素的**结构**。

  HTML 运行在浏览器上，由浏览器来解析。

+ 根**标签**`<html></html>`，结构都放在根标签中。

  在根标签里面的就是子标签`<html><head>...</head></html>`

  ~~~html
  <html> 
      <!-- html是根标签，其他的是它的子标签 -->
      <head>
          <title>...</title>
          <!-- title标签中放的是网页的名称 -->
      </head>
      <body>
          <!-- body中放的是网页的主体 -->
      </body>
  </html>
  <!-- 网页是由一个个标签组成的，每个标签有开始有结束 -->
  ~~~
